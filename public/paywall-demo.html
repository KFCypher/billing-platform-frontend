How It Works

  1. User logs in with their email on your website
  2. Your website calls BillingPaywall.verifySubscription(email)
  3. The API checks if the user has an active subscription
  4. Protected content is shown or hidden based on subscription status

Implementation Example:

<!-- 1. Include the paywall script -->
<script src="http://localhost:3000/paywall.js"></script>

<!-- 2. Initialize with your API key -->
<script>
    BillingPaywall.init('pk_test_your_api_key_here');
</script>

<!-- 3. Protect content -->
<div id="premium-content" style="display: none;">
    Your premium content here...
</div>

<script>
    // Get user email from your auth system
    const userEmail = getCurrentUserEmail();
    
    // Protect the content
    BillingPaywall.protect('#premium-content', userEmail, {
        upgradeUrl: '/pricing',
        upgradeMessage: 'Upgrade to Premium to access this content'
    });
</script>
